# Mini URL Shortener API

## Description

A REST API that shortens long URLs and redirects to the original URL.

## Tech Stack

- Node.js
- Express.js
- MongoDB
- NanoID for unique short code generation

## Setup Instructions

```bash
git clone https://github.com/AnuragKush2527/URL-Shortener.git
cd URL-Shortener
npm install

## Run the server
npm start
```
